# Internship Project

## How to run

### Server

Run on express server

```
cd server/
yarn install
node index.js
```

Go to localhost:4000

**Then on a different console.**

### Client

```
cd client/
yarn install
yarn run start
```

Go to localhost:3000


### Important

Either use `.env` and `dotenv`
Or
Add a `keys.js` file under `config` with the same keys as in `.env`
